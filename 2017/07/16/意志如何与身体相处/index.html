意志如何与身体相处

Posted on 2017-07-16   |     |   次阅读

最近看了很多健身视频，常常谈到用意念／意志控制身体，能指挥身体走向想要的方向，又联想到除了运动之外，生活工作中我们的意志对于身体的暗示也对身体产生非常重要的作用。让我相信身体和意志是分离的，又是浑然一体的，如果能控制意志与身体和谐相处，那很多你觉得的疼痛或者快感都能在身体上得到感受。

健身讲求意念力，其实就是意志控制神经，比如你身体不动，但你可以让你的二头肌发力，然后你就可以看到肌肉的活动，那么在健身中，每做一个动作都要想着那里的肌肉要发力，才会很有效果。如果意志与身体完全没有联系，或者身体主导了意识，你就会感受不到身体的活力，身体锻炼的反馈成果也很难达到你的期望。

上面是健身中，再想想平时意志力对身体有什么影响？如果有天起床，我们觉得没有睡好，身体没有力气，那这一天很可能就是昏昏沉沉，尽管可能身体和平时并没什么两样，但它真的相信了大脑传达的信息，今天都不准备怎么出力。相反，如果我们生病了，但仍相信自己的身体很棒，不会有太多消极想法，继续照常锻炼活动，可能过两天我们就好了，也不会受什么身体病了的影响。

这样的例子还有很多。

鲁迅曾经在《看镜有感》中写道：

无论从那里来的，只要是食物，壮健者大抵就无需思索，承认是吃的东西。惟有衰病的，却总常想到害胃，伤身，特有许多禁条，许多避忌；还有一大套比较利害而终于不得要领的理由，例如吃固无妨，而不吃尤稳，食之或当有益，然究以不吃为宜云云之类。但这一类人物总要日见其衰弱的，因为他终日战战兢兢，自己先已失了活气了。

这应该就是积极的信号带给身体的积极影响吧，是为什么医生都会说压力能让人生病，生病时要保持愉悦的心情的原因吧。每个人都需要学会和身体相处，让意志控制身体，利用身体，而不是让身体告诉意志今天适合做什么不适合做什么。长此以往，我相信你会更相信你的身体，更明白如何与身体和谐相处。

当然需要注意的是，我们身体的机能会对外界的刺激作出反应，如果超负荷使用身体太多，它会生病，会疲惫，我们也要学会去感受它，不是一味的去滥用身体，这是所谓的和谐相处，有来有往。
