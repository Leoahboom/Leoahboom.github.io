《梁思成与他的时代》

Posted on 2017-05-29   |     |   次阅读

《梁思成与他的时代》是建筑师朱涛先生所写，本书的优点在于紧密结合梁思成所生活的时代背景来探讨他的作品思想的脉络和背后可能的原因，有助于后人更了解真实的梁思成和他所经历的抗战、解放前后的时代所发生的事，对梁先生的评价也是有褒有贬，相对客观。在当代媒体和群众一味歌颂“梁陈方案”的背景下，本书所描写的建国之初对首都规划方案部分显得尤为珍贵。

而读罢这些关于梁先生的史实和探讨后，我看到的是世上无完人的残酷现实，无论多么伟大的人类，亦逃脱不了思维眼界的束缚（伟人眼界宽广，但仍有极限），加上近日柯洁大战Alpha Go，号称它和和人类对于围棋的理解完全不在一个层次，我的理解是Alpha Go为人类所创造，我不知道人工智能和机器学习能多大程度脱离人类思维自由发展，但我片面的理解可能是受过围棋教育的人和程序员／工程师对围棋的思路和理解不同，造成Alpha Go的独特。其实这里想说的就是每个人都有一套自己的思维世界，但没有一套是完美无缺的，后世可能都可以推敲某个伟人的决策与思路，但没人能改变人类思维片面性这一客观事实。既然无法改变，我们就需要接受，不断修正，希冀能走向更好的方向。

《梁思成与他的时代》从梁先生留学美国开始写起，西方教育体系和知识内容让梁先生最初成为建筑布杂体系的拥趸（偏古典主义），后又受柯布西耶等现代主义思想影响，这些西方思想也随着梁先生返回祖国被应用到了当时的中国建筑研究中。自梁先生回国，他经历了抗战时期、内战、建国、新中国建设、文革等一系列重大历史事件，他的思想和学术成就也不断在变化，从最初营造学社的中国古代建筑研究高峰到最后的思想斗争岁月，鲜有学术创作。梁先生的一生曾无比光鲜，坚持的思想也曾无人采纳，苦不堪言。人在时代的大背景下，走的是一条什么路，恐怕自己早已看不清，只能等待后世的梳理和评价。

最后再说说“梁陈方案”，如今北京城的拥堵和古迹的消失，成为人们追忆历史的借口，对历史的假设就是对如今的不满，仿佛当初采取了“梁陈方案”，今天的北京就可以畅通无阻。然而历史无法假设，再来一遍“梁陈方案”还是不会被采纳，书中探讨了“方案”在设计方面的一些不合实际，不接地气，也陈述了当时政府在面对国内外政治经济等压力下不得不作出的一些决定，当时对于新首都的规划不只是“梁陈方案”一种，还有许多大家名士提出自己的想法，而最终采取的方式也是各方面折衷而来，有时候我觉得如今的局面可能是我们走得太快而造成的。

书的结尾又提到当今中国建筑发展没有自己的灵魂，什么样的建筑都可以建造出来，确实如此，在很多方面我们的国家和人民没有什么“主见”，而什么样的建筑才是中国建筑或者我们其实无需自己的特色，又及中国建筑应该向何种方向发展？这些大问题，似乎没人能解答，也没人能起到控制作用，任由时代的车轮碾过，好与坏只能留与后人评论。
